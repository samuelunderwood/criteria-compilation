Some editorial measures were taken while transcribing the values from their original sources. These changes were intended to improve filterability and avoid an unnecessarily long list of room type labels which differed only in syntax/spelling while referring to the same room function. Work is currently underway to bring the database in alignment with these formatting norms.

Formatting Norms

1. Plural noun room types

All room type labels have been transcribed in plural form.

For example:

'Concert halls' and not 'Concert hall'

2. Capitalization

Categories Use Title Case (every word capitalized)

Room type labels use sentence case (only the first word capitalized)

3. Nomenclature alignment

Where possible, room type labels have been tweaked to use similar nomenclature.

For example:

"General classrooms" and "schoolrooms" have been transcribed simply as "classrooms"

"Cinemas" and "movie theaters" have been transcribed simply as "movie theaters"

4.Qualifiers/subtypes

Many resources delineate qualifiers for specific sub-categories of a room use. Common qualifiers include:

  • Subjective size (small, large)
  • Objective size/volume/seating capacity (ranges in m3 or ft3, or number of seats)
  • Amplified vs. unamplified speech
  • Quality (industry, premium, loud vs. quiet)
  • Class (Class A/Class B/Class C/Class D)

Meanwhile, other references make no such distinction. To make it easier to see criteria for a common group, qualifiers are listed separately in the 'subtype' column
